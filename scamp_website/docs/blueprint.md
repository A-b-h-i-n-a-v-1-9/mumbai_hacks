# **App Name**: Scamp: AI Scam Bodyguard

## Core Features:

- Real-time Scam and Deepfake Detection: Screens audio, video, images, and text for impersonation and scam scripts in real time, leveraging AI tool for pattern recognition.
- Risk Score Engine: Generates a risk score (0-100) for each request with labels, providing a clear assessment of potential scam risk.
- Explainable AI: Breaks down why something looks risky in everyday language, offering transparency and user understanding. Uses AI tool for determining risk explanations.
- Auto Cybercrime Report Drafts: Automatically drafts detailed cybercrime reports with evidence that can be submitted to cyber cells and portals.
- Adaptive Learning Agent: Improves scam pattern recognition based on user feedback, adapting to regional and language-specific scams.
- Chrome Extension for Web Protection: Flags risky websites, payment links, and web-based chats in real time using a browser extension.
- Telegram Integration: Provides seamless integration with Telegram to scan messages and media.

## Style Guidelines:

- Background: Near-black (#020617) for a dark, fintech-inspired look.
- Card background: Gradient from #020617 to #111827 with glassmorphism effect.
- Primary CTA accent: Vibrant green (#22C55E) for a strong call to action.
- Secondary accent: Light blue (#38BDF8) for highlights and micro-interactions.
- Heading font: 'Space Grotesk' (sans-serif) for bold headings.
- Body font: 'Inter' (sans-serif) for clear and readable text.
- Note: currently only Google Fonts are supported.
- Use simple, clean icons to represent features and benefits.
- Full-width, single-page, scrollable landing page with a sticky navbar.
- Subtle hover states and transitions for interactive elements.
